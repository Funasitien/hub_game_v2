Documentation sur le code Morse

Le code Morse est un système d'encodage de caractères textuels en séquences de signaux, telles que des signaux sonores, des signaux lumineux ou des impulsions électriques, utilisé pour transmettre du texte par des moyens tels que la télégraphie électrique.

Chaque lettre et chiffre est représenté par une séquence spécifique de points (·) et de tirets (-). Voici la correspondance des lettres de l'alphabet et des chiffres en code Morse :
